## Stopwatch - v1.2
A stopwatch timer that was inspired off of [Jcog](https://twitch.tv/jcog)'s timer, but with no crazines. I just wanted it to be custom :)
### Features:
40s tps, custom fonts, and more!
### Changelog

_       _New Settings menu with a font changer. The display has a "forced monospace" property which will eliminate the jitteriness of otherwise professional fonts, making them easier to read, and better in the eyes\
_          _**Default** - Fancy Cat PX. A tabular font suitable for a clock!
_          _**System** - An OS-specific setting. iOS/Mac will use San Francisco, Android will use Roboto, and Windows will use Segoe UI
_          _**Custom** - Upload your own TTF, OTF, or WOFF. **Note- You may need to refresh the page once you've uploaded a file!**
